# CRUD operation using MERN

## Technologies Used:

- React - Frontend
- NodeJs - Backend
- ExpressJs - Backend
- MongoDB - Database

### Installation Process

1. Clone the Repository
```
git clone https://github.com/santhoshpandi/crud-mern.git
```

2. Start the Server (Backend)
```
cd backend
npm run dev
```

3. Start the Client (Frontend)
```
cd frontend
npm run dev
```
